The correct way to NPM Install

The reason I say the correct way is because you should be doing npm install separately for every project, and managing your npm imported dependencies in the package.json for each project. This is also very important for hosting a node.js app as well.

  • Step 1
  • Step 2
  • Step 3
  • Step 4
  • Video

  1. Create a folder representing your node.js project

  2. Navigate to that node.js project directory using your terminal/bash

  3. Run npm init in your terminal/bash and press enter all the way through

  4. I will be importing the 'request' dependency from NPM to my project using this command: npm install request --save
