  1. Create a folder, and inside of that folder create a .html and a .js file

  2. Put contents in the .html file and the .js file.
    a. For the contents in the .html file, there needs to be a script tag that connects the javascript file to the html file. Place this script tag right above the closing of the body tab. The "src" field in the script tag represents the path to the connecting javascript file. This is how my html file looks:

  3. Get the path to that .html. I am using the "atom" text editor, which allows me to copy the path to a file.

  4. Paste the path to that .html into your browser. I am pasting into my Google Chrome Browser.

  5. These instructions are strictly for Google Chrome. Right click on the browser, and open up Google Inspect. Go to the console tab, and you should see the console log from your javascript file
