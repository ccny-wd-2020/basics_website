  1. Once you have created your github repository, copy the ssh url. It should look like: git@github.com:{github_username}/{project_name}

  2. Open up your terminal (mac) or git bash (windows)

  3. Commands are as follows

    • cd ~/directory/to/where/your/work/is
    • git init
    • git remote add origin {github_repository_url}
    • git checkout -b main
    • git add .
    • git status
    • git commit -m 'adding changes'
    • git push --set-upstream origin main

  4. This is what your github repository should look like.
