Pushing Site to Host

I used namecheap for my host, so the instructions here will cover the instructions for hosting with Namecheap

  • Step 1
  • Step 2
  • Step 3 - Accessing your code via the browser
  • Step 4 - Accessing your code via the terminal

  1. Go to Namecheap.com via this link to get the shared hosting and the domain name for a total of $27.30

  2. You will get 2 emails. The 2nd one should contain your Hosting Account Details. The important ones here are the cPanel username and the cPanel password

  3. Accessing your code via the browser

    1. Go to your website's cPanel i.e. "https://mywebsite.com/cpanel"
    2. Enter your cpanel username and password
    3. Once in your cPanel, go to "File Manager" under the "Files" section
    4. Double click on the "public_html" directory and that is where all of your html, css & js files will go
    5. With your main/root/home index file being named "index.html", put it in the "public_html" directory along with the rest of your project files, and your project should be connected. There might be some file path problems in terms of the css and js file paths in your html files. Do those adjustments.

  4. Accessing your code via the terminal

    1. Go to your Hosting Account Details email, and fine the Server Hostname, the cPaner Usernamd and cPanel Password
    2. Type this command in the terminal ssh cPanelUsername@serverHostname -p21098. Mine looks like ssh webddrlg@server290.web-hosting.com -p21098. Type in your password once prompted.
    3. Once signed in, navigate to the "public_html" directory. You can add and adjust code from here using the terminal editor with vi
    4. With your main/root/home index file being named "index.html", put it in the "public_html" directory along with the rest of your project files, and your project should be connected. There might be some file path problems in terms of the css and js file paths in your html files. Do those adjustments.
