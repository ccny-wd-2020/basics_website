  1. Once you have created your github repository, click the green "Code" button and copy the ssh url. It should look like: git@github.com:{github_username}/{project_name}

  2. Open up your terminal (mac) or git bash (windows)

  3. Commands are as follows

    • cd ~/directory/to/where/your/work/is
    • git init
    • git remote add origin {github_repository_url}
    • git checkout -b main
    • git add .
    • git status
    • git commit -m 'adding changes'
    • git pull origin main --allow-unrelated-histories
    • git push --set-upstream origin main

  4. At the git pull origin main --allow-unrelated-histories step in step3, to get out of the editor, please "escape", and then ":wq", and then enter. View the bottom left of the editor in the image below to see exactly what's done to get out of there.
